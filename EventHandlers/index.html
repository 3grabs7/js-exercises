JavaScript Exercises

Event Handling

Exercise 1

Hantera event för button så browsern gör en alert med värdet i input som message.

Exercise 2

Hantera event för inputs och buttons så att browsern först gör en prompt med värdet i Question som message när Go klickas och sedan sätter användarens svar i Answer. Gör även så clear nollställer både Question och Answer.

Lägg till en alert om validering ifall frågan ej innehåller ett och endast ett frågetecken placerat på slutet.

Exercise 3

Hantera event för input så att endast '1' och '0' tillåts att skrivas in.
Om något annat tecken används skall inputen ej skriva ut detta.
(Kom ihåg att backspace måste fungera som vanligt)

När input value är 5 tecken långt ska broswern alerta att vi är klara. När användaren har klickat ok ska input disablas.

Exercise 4

Hantera event så resultatet av operand 1 och operand 2 skrivs ut i result när en av button med operatorer hovras.
När vi slutar hovra skall result nollställas.

I operanderna tillåts endast siffror och leading zeroes skall undvikas.
(Igen, kom ihåg att backspace måste fungera som vanligt)
Säkra även inputs så att vi ej kan hacka oss till att ha endast nollor kvar.
ex. om vi skriver 6700 och tar bort 67 ska vi ej ha 00 kvar.
